# HQ Investment Billing - Landing Page

This is the landing page for the HQ Investment Billing System.

## Features
- Modern, responsive UI with soft blue/white theme
- Framer Motion animations
- Tailwind CSS styling
- Contact form integration
- Mock subscription API
- JSON-based temporary storage

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure
- `src/app`: App Router pages and API routes
- `src/components`: Reusable UI components
- `src/styles`: Global CSS and Tailwind configuration
- `data`: JSON database storage
- `src/lib`: Utility functions and database helpers
